## ✨ Возможности

- 🚀 **Работа Telegram без VPN** — обход блокировок через WebSocket-соединения
- 🖥️ **Нативное macOS приложение** — иконка в трее, не занимает место в Dock
- 💪 **Полная поддержка Apple Silicon** — оптимизировано для M1/M2/M3/M4/M5
- 🔧 **Простая настройка** — все параметры через GUI или конфигурационный файл
- 📊 **Статистика в реальном времени** — отслеживайте трафик и состояние подключений
- 🔄 **Автоматическое восстановление** — fallback на TCP при проблемах с WebSocket
- ⚡ **Оптимизированная скорость** — пул соединений и health-check для лучших IP

## 📥 Установка

### Вариант 1: Готовое приложение (рекомендуется)

1. Скачайте последнюю версию **TG-WS-Proxy-ARM64-*.dmg** из [раздела релизов](https://github.com/Maxim-szh/tg-ws-proxy-mac/releases)
2. Откройте скачанный `.dmg` файл
3. Перетащите **TG WS Proxy.app** в папку **Программы**
4. Запустите приложение из папки **Программы**
5. При первом запуске: нажмите правой кнопкой мыши → **Открыть** (разрешить)

### Вариант 2: Из исходников (для разработчиков)

```bash
# Клонируйте репозиторий
git clone https://github.com/Maxim-szh/tg-ws-proxy-mac.git
cd tg-ws-proxy-mac

# Создайте виртуальное окружение (рекомендуется)
python3 -m venv venv
source venv/bin/activate

# Установите зависимости
pip install -r requirements-mac.txt

# Запустите приложение
python app/macos.py
```

## 🚀 Использование

1. **Запустите** TG WS Proxy — иконка появится в меню-баре (справа сверху)
2. **Откройте** Telegram Desktop
3. **Настройте прокси**(автоматически):
   - Перейдите в **Настройки** → **Продвинутые** → **Тип подключения** → **Прокси**
   - Добавьте новый прокси:
     - **Тип:** `SOCKS5`
     - **Сервер:** `127.0.0.1`
     - **Порт:** `1080`
     - Логин/пароль оставьте пустыми
4. **Готово!** Telegram начнет работать через ускоренные WebSocket-соединения

## ⚙️ Настройка

Через иконку в трее доступны следующие действия:

| Действие | Описание |
|----------|----------|
| 🔗 **Открыть в Telegram** | Автоматически открывает Telegram с настройками прокси |
| 🔄 **Перезапустить прокси** | Перезапускает прокси-сервер (полезно после изменения настроек) |
| ⚙️ **Настройки...** | Открывает окно для изменения порта и DC-серверов |
| 📂 **Открыть логи** | Показывает файл с логами работы |
| 🚪 **Выход** | Завершает работу приложения |

## 🔧 Конфигурация

Все настройки хранятся в файле:

```
~/Library/Application Support/TgWsProxy/config.json
```

## 📋 Логи

Логи приложения находятся по пути:

```
~/Library/Application Support/TgWsProxy/proxy.log
```
